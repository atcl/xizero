Sun Raster File Format

Header:

typedef struct _SunRaster
{
  DWORD MagicNumber;      //Magic (identification) number = 0x956aa659h
  DWORD Width;            //Width of image in pixels (always multiple of 2 bytes, padded if necessary) 
  DWORD Height;           //Height of image in pixels
  DWORD Depth;            //Number of bits per pixel, 24 or 32
  DWORD Length;           //Size of image data in bytes
  DWORD Type;             //Type of raster file ( 0000h,0001h Default; 0002h RLE-encoded, FFFFh Experimental including transparency)
  DWORD ColorMapType;     //Type of color map (Deprecated) = 0
  DWORD ColorMapLength;   //Size of the color map in bytes (Deprecated) = 0
} SUNRASTER;

Data:

Default:
RGB or ARGB encoding

RLE:
packet: 
1.Byte (flag)  = 0x80h
2.Byte (count) = 0x01h ... 0xFFh
3.Byte (value) = 0x00h ... 0xFFh
Exceptions:
2.Byte == 0x00h => single byte of value 0x80h
1.Byte != 0x80h => uncompressed byte value
